SDLPAL
======

A simplified version of [SDLPAL](https://github.com/sdlpal/sdlpal) featuring a minimal footprint of 10k LOC.

This project provides a highly approachable codebase tailored for experimentation, hacking, and porting.

Only support the [baiyou version of pal98](https://archive.org/download/pal1_1995/XJ98rqp.zip).

With:
* C99
* SDL3
* libiconv (for gbk to unicode)
* freetype2 (for ttf)
* adplug  (for midi)
